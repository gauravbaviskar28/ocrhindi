STEPS :
1.Create a virtual env in python and activate it. 
2.Come to the same directory as the manage.py file.
3. type the following command in cmd to install all the packages : 
    pip install -r requirements.txt
4. After succesful installation run the following command to set up the server :
    python manage.py runserver
5. That will create a development server at localhost:8000
6. ocr/ocrcode.py file is the file where the model is implemented.
7. To improve the UI go to templates folder.
